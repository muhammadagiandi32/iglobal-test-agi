### Menjalankan service

-   bash
    ```sh
    composer install
    ```
    -   bash
    ```sh
    npm install
    ```
-   bash
    ```sh
    php artisan migrate:fresh --seed
    ```
    -   bash
    ```sh
    php artisan serve
    ```
    -   bash
    ```sh
    npm run dev
    ```
    _Pastikan service berjalan pada url `http://localhost:8080`._
